Personal Projects

inHouse

(formerly 'The Snooty Fox Menu App')

A hybrid app created from the hardships of COVID-19.
A food app that allows customers to order take-out for
curbside pickup, and for ordering food and drinks while
with a restaurant. This would alleviate some converns
regarding social distancing and the spread of COVID-19.
This app follows the design of popular ordering apps such
as Skip the Dishes and Uber Eats. It was created this way
for ease of use for users, giving them a farmilar interface.

Languages & Technology Used:

  • mySQL (menu, accounts, orders)
  • Ionic (for Hybrid Wrapper)
  • Java (for Android)
  • HTML & CSS
  • JavaScript
  • jQuery
  • React
  • PHP

inHouse Manager

A hybrid app to be used in accordance with Inhouse regular
app. This app allowed the staff of a restaurant to manage
the orders made by customers.

Features

  • View orders recieved
  • Mark orders as active
  • View all active orders
  • View filtered active orders
  • Mark orders as completed
  • Search all orders (received, active, completed)
  • Summary of orders based on filtered criteria:
    • Account
    • Order #
    • Date
    • Table #
    • Total price
  • GUI that allows managers to easily edit database
    information
    • Edit menu item
    • Add new menu item
    • Delete menu item
    • Upload images for menu items
